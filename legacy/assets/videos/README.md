# Video Assets

This directory contains video clips for the Elvora Consulting website.

## Supported Formats
- MP4 (recommended for web)
- WebM (for better compression)
- MOV (if needed for specific purposes)

## Recommended Video Specifications
- **Resolution**: 1920x1080 (Full HD) or 1280x720 (HD)
- **Aspect Ratio**: 16:9 (landscape) or 9:16 (portrait for mobile)
- **Duration**: Keep videos concise (30 seconds to 2 minutes for web)
- **File Size**: Optimize for web (under 10MB when possible)

## Usage Examples
- Testimonial videos
- Service demonstrations
- About Melissa video
- Process walkthroughs
- Training snippets

## Optimization Tips
- Compress videos for web delivery
- Consider creating multiple versions (desktop/mobile)
- Add captions/subtitles for accessibility
- Use descriptive filenames (e.g., `melissa-introduction.mp4`)
